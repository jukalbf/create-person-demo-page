# Demo Page para criação de entidade Pessoa
## Descrição:
Demo page criada para ser usada como demonstração gráfica com projeto backend.

>[!WARNING]
>Ainda em desenvolvimento!!
## Funcionalidade padrão:
Formulário para adicionar dados de uma nova pessoa com *nome*, *email*, *CPF* e *assinatura digital*. Assim como a listagem de todas as pessoas já registradas, que atualiza assim que uma nova é adicionada.
## Tecnologias:
- HTML
- CSS
- JS
## Modo de uso:
Caso queira usar para testar o seu projeto, você pode alterar o código de acordo com as suas necessidades.
#### Classes de estilo padrão:
- `.registe-container`: `div` geral com os inputs de dados e o canvas;
- `.personal-data`: `div` apenas com os inputs e o `button`;
- `.input`: classe das tags `input`;
- `.digital-signature-container`: `div` container da assinatura digital;
- `.digital-signature-canvas`: class do `canvas`;
- `.data-list-container`: classe da `ul`;
- `.data-info-container`: classe das tags `li`;
- `.person-name`: classe da tag `h2` dos cards que contem as informações dos usuários; 
- `.person-email`: classe da tag `span` com valor do email nos cards que contem as informações dos usuários;
- `.person-cpf`: classe da tag `span` com valor do CPF nos cards que contem as informações dos usuários;
- `.signature-img`: classe da tag `img` no card que mostra a imagem da assintura digital.
